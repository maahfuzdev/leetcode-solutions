## `$ cat style_guide.py`

```python
# ============================================================
# Problem    : Two Sum (#1)
# Difficulty : Easy  |  Category: Array
# ------------------------------------------------------------
# Approach   : HashMap — store seen values, check complement
# Time       : O(n)
# Space      : O(n)
# ============================================================

class Solution:
    def twoSum(self, nums: list[int], target: int) -> list[int]:
        seen = {}
        for i, num in enumerate(nums):
            complement = target - num
            if complement in seen:
                return [seen[complement], i]
            seen[num] = i
```

> Every solution follows this exact format — problem header, approach in plain English, complexity analysis, clean code.
